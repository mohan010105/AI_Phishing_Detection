### 💡 The Inspiration
Phishing attacks are becoming increasingly sophisticated, exploiting urgency and using hidden vectors like QR codes or brand spoofing to bypass traditional security filters. PhishGuard was created to bridge this gap, translating raw security data into clear, natural language explanations and actionable steps for everyday users.
